LinguaWatch

Privacy

LinguaWatch is a browser extension that helps you learn language while watching captioned videos on YouTube. This page describes what data is involved in the beta.

Video captions. The extension reads caption text from the page so it can choose phrases and build lessons. It does not upload your full watch history to LinguaWatch servers (there is no LinguaWatch server in this beta).

OpenAI. To generate translations, teaching content, and optional audio, the extension sends text (the selected phrase and related lesson content) to OpenAI’s API using an API key configured in the extension. That means OpenAI processes those requests under their policies.

Settings. Extension preferences (such as on/off for the current site) may be stored in your browser via Firefox sync storage if you use sync.

Contact. For beta questions, use whatever channel the developer shared with you.
